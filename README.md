
# Tazk

Final project...

### Github

git clone https://github.com/nachgrandi/Tazk.git

## Setup

These environment variables are required:

* PORT
* JWT_SECRET
* MONGODB_URI
* MONGODB_USER
* MONGODB_PASSWORD

## Running

`npm i` for installation of node_modules

`npm run dev` for execution
